# 音楽リズムゲーム 🎵

Webブラウザで遊べるリズムゲームです。音楽に合わせてノーツをタップして高スコアを目指しましょう！

## 機能

- **リアルタイムリズムゲーム**: 音楽に合わせてノーツが降ってきます
- **4レーン**: D、F、J、Kキーを使用
- **スコアシステム**: タイミングによってPerfect、Great、Good、Missが判定されます
- **コンボシステム**: 連続でヒットさせるとスコアボーナス
- **視覚エフェクト**: ヒット時のパーティクル効果

## 操作方法

- **D**: 左から1番目のレーン
- **F**: 左から2番目のレーン  
- **J**: 左から3番目のレーン
- **K**: 左から4番目のレーン

## ファイル構成

```
/
├── index.html          # メインのHTMLファイル
├── game.js            # ゲームロジック
├── assets/
│   └── audio/
│       ├── hentaisong.mp3
│       └── hentaisong02.mp3
├── public/
└── README.md          # このファイル
```

## 遊び方

1. `index.html`をWebブラウザで開く
2. 「ゲーム開始」ボタンをクリック
3. 音楽に合わせてノーツがヒットゾーン（白い線）に来たタイミングでキーを押す
4. より正確なタイミングでヒットさせて高スコアを目指す

## 判定について

- **Perfect**: ±15px以内 - 300点 + コンボボーナス
- **Great**: ±30px以内 - 200点 + コンボボーナス  
- **Good**: ±50px以内 - 100点 + コンボボーナス
- **Miss**: 判定範囲外 - 0点、コンボリセット

## 音楽ファイルについて

`assets/audio/`フォルダにMP3ファイルを配置してください。ゲームは以下の順序でファイルを読み込みます：

1. `assets/audio/hentaisong.mp3`
2. `assets/audio/hentaisong02.mp3`
3. `hentaisong.mp3` (ルートディレクトリ)

## 技術仕様

- **HTML5 Canvas**: ゲーム描画
- **Web Audio API**: 音楽再生
- **JavaScript ES6**: ゲームロジック
- **CSS3**: UI styling

## ブラウザ対応

モダンブラウザ（Chrome、Firefox、Safari、Edge）で動作します。
